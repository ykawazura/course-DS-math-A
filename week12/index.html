    第12回：重積分

    12.1 1変数の積分（復習）

    1変数関数 \(f(x)\) の定積分

    \[ \int_a^b f(x)\, dx \]

    は、\(y = f(x)\) と \(y = 0\)（\(x\) 軸）の間の符号付面積を表す。\(f(x) > 0\) の部分は正、\(f(x) < 0\) の部分は負として面積を計算する。

    12.2 2変数の場合

    2変数関数 \(f(x, y)\) の重積分（二重積分）

    \[ \iint_D f(x, y)\, dx\, dy \]

    は、曲面 \(z = f(x, y)\) と \(z = 0\)（\(xy\) 平面）の間の符号付き体積を表す（\((x, y) \in D\)）。

    12.3 具体的な計算方法

    重積分を具体的に計算するには、フビニの定理を用いて逐次積分（累次積分）に帰着させる。

    (1) \(x\) 平面でスライス

    \(x\) を固定し、\(y\) 方向に積分して断面積 \(S(x)\) を求める。

    \[ S(x) = \int_{a(x)}^{b(x)} f(x, y)\, dy \]

    これを \(x\) 方向に積分すると体積が得られる。

    \[ \iint_D f(x, y)\, dx\, dy = \int_{x_0}^{x_1} S(x)\, dx = \int_{x_0}^{x_1} \left( \int_{a(x)}^{b(x)} f(x, y)\, dy \right) dx \]

    (2) \(y\) 平面でスライス

    \(y\) を固定し、\(x\) 方向に積分して断面積 \(T(y)\) を求める。

    \[ T(y) = \int_{c(y)}^{d(y)} f(x, y)\, dx \]

    これを \(y\) 方向に積分すると体積が得られる。

    \[ \iint_D f(x, y)\, dx\, dy = \int_{y_0}^{y_1} T(y)\, dy = \int_{y_0}^{y_1} \left( \int_{c(y)}^{d(y)} f(x, y)\, dx \right) dy \]

    どちらのスライスで計算した結果も同じであるというのがフビニの定理である。

    12.4 フビニの定理

    フビニの定理

    \(f(x, y)\) が領域 \(D\) 上で連続であるとき、重積分は逐次積分（累次積分）に変換できる。

    \[ \iint_D f(x, y)\, dx\, dy = \int_{x_0}^{x_1} \left( \int_{a(x)}^{b(x)} f(x, y)\, dy \right) dx = \int_{y_0}^{y_1} \left( \int_{c(y)}^{d(y)} f(x, y)\, dx \right) dy \]

    逐次積分とは、多重積分を一変数の積分の繰り返しに帰着させる方法である。内側の積分では一方の変数を固定して積分し、その結果を外側の変数で積分する。

    • \(x\) で先に外側を積分する場合：内側で \(y\) を \(a(x)\) から \(b(x)\) まで積分し、外側で \(x\) を \(x_0\) から \(x_1\) まで積分する。
    • \(y\) で先に外側を積分する場合：内側で \(x\) を \(c(y)\) から \(d(y)\) まで積分し、外側で \(y\) を \(y_0\) から \(y_1\) まで積分する。

    フビニの定理は、積分の順序を交換しても結果が等しいことを保証する。

    12.5 重積分の性質

    (1) 線形性

    \[ \iint_D \bigl[f(x, y) \pm g(x, y)\bigr]\, dx\, dy = \iint_D f(x, y)\, dx\, dy \pm \iint_D g(x, y)\, dx\, dy \]

    (2) 定数倍

    \[ \iint_D k\, f(x, y)\, dx\, dy = k \iint_D f(x, y)\, dx\, dy \]

    (3) 領域の分割

    領域 \(D\) が \(D = D_1 \cup D_2\)（\(D_1 \cap D_2\) が面積 \(0\)）と分割されるとき、

    \[ \iint_D f(x, y)\, dx\, dy = \iint_{D_1} f(x, y)\, dx\, dy + \iint_{D_2} f(x, y)\, dx\, dy \]

    (4) 長方形領域で変数分離できる場合

    \(D\) が長方形 \(a \le x \le b\), \(c \le y \le d\) で、\(f(x, y) = g(x)\, h(y)\) と変数分離できるとき、

    \[ \iint_D f(x, y)\, dx\, dy = \left( \int_a^b g(x)\, dx \right) \left( \int_c^d h(y)\, dy \right) \]

    12.6 積分の書き方に関する注意

    \[ \int_a^b \left( \int_c^d f(x, y)\, dx \right) dy \]

    は

    \[ \int_a^b dy \int_c^d dx\, f(x, y) \]

    とも書く。ここで

    • \(\displaystyle\int_c^d dx\) を「\(x\) に関する積分作用素」
    • \(\displaystyle\int_a^b dy\) を「\(y\) に関する積分作用素」

    と見ることができる。

    12.7 例題

    例題1（教科書 p.218）

    \(\displaystyle\iint_D xy\, dx\, dy\), \(D = \{(x, y) \mid 0 \le x \le 1,\; 0 \le y \le 2\}\) を求めよ。

    解

    \(D\) は長方形で \(f(x, y) = xy\) は変数分離できるから

    \[ \iint_D xy\, dx\, dy = \left( \int_0^1 x\, dx \right) \left( \int_0^2 y\, dy \right) \]
    \[ = \left[ \frac{x^2}{2} \right]_0^1 \cdot \left[ \frac{y^2}{2} \right]_0^2 = \frac{1}{2} \cdot 2 = 1 \]
    別解（逐次積分）
    \[ \int_0^2 \left( \int_0^1 xy\, dx \right) dy = \int_0^2 y \left[ \frac{x^2}{2} \right]_0^1 dy = \int_0^2 \frac{y}{2}\, dy = \left[ \frac{y^2}{4} \right]_0^2 = 1 \]

    例題2（教科書 p.229）

    \(\displaystyle\iint_D (x + 4y + 2)\, dx\, dy\), \(D = \{(x, y) \mid 1 \le x \le 2,\; x \le y \le 2x\}\) を求めよ。

    解

    \(x\) で外側を積分する。

    \[ \int_1^2 \left( \int_x^{2x} (x + 4y + 2)\, dy \right) dx \]

    内側の積分を計算する。

    \[ \int_x^{2x} (x + 4y + 2)\, dy = \left[ (x + 2)y + 2y^2 \right]_{y=x}^{y=2x} \]
    \[ = (x + 2) \cdot 2x + 2(2x)^2 - (x + 2) \cdot x - 2x^2 \]
    \[ = 2x^2 + 4x + 8x^2 - x^2 - 2x - 2x^2 = 7x^2 + 2x \]

    外側の積分を計算する。

    \[ \int_1^2 (7x^2 + 2x)\, dx = \left[ \frac{7}{3}x^3 + x^2 \right]_1^2 \]
    \[ = \frac{56}{3} + 4 - \frac{7}{3} - 1 = \frac{49}{3} + 3 = \frac{58}{3} \]

    例題3（教科書 p.230）

    \(\displaystyle\iint_D y\, dx\, dy\), \(D = \{(x, y) \mid x + y \ge 1,\; x^2 + y^2 \le 1\}\) を求めよ。

    解

    領域 \(D\) は単位円 \(x^2 + y^2 \le 1\) のうち直線 \(x + y = 1\) より上側の部分である。

    \(x\) で外側を積分する。\(x + y = 1\) より \(y = 1 - x\)、\(x^2 + y^2 = 1\) より \(y = \sqrt{1 - x^2}\)（上半分）。\(x\) の範囲は \(0 \le x \le 1\)。

    \[ \int_0^1 \left( \int_{1-x}^{\sqrt{1-x^2}} y\, dy \right) dx \]
    \[ = \frac{1}{2} \int_0^1 \left[ y^2 \right]_{1-x}^{\sqrt{1-x^2}} dx = \frac{1}{2} \int_0^1 \left( 1 - x^2 - (1 - x)^2 \right) dx \]
    \[ = \frac{1}{2} \int_0^1 \left( 1 - x^2 - 1 + 2x - x^2 \right) dx = \frac{1}{2} \int_0^1 (2x - 2x^2)\, dx \]
    \[ = \int_0^1 (x - x^2)\, dx = \left[ \frac{x^2}{2} - \frac{x^3}{3} \right]_0^1 = \frac{1}{2} - \frac{1}{3} = \frac{1}{6} \]

    演習問題

    演習1（教科書 p.218）

    \(\displaystyle\iint_D x^2 y\, dx\, dy\), \(D = \{(x, y) \mid 0 \le x \le 1,\; 0 \le y \le 1\}\) を求めよ。

    解答

    \(D\) は長方形で \(f(x, y) = x^2 y\) は変数分離できるから

    \[ \iint_D x^2 y\, dx\, dy = \left( \int_0^1 x^2\, dx \right) \left( \int_0^1 y\, dy \right) = \left[ \frac{x^3}{3} \right]_0^1 \cdot \left[ \frac{y^2}{2} \right]_0^1 = \frac{1}{3} \cdot \frac{1}{2} = \frac{1}{6} \]

    演習2（教科書 p.230）

    \(\displaystyle\iint_D (x + y)\, dx\, dy\), \(D = \{(x, y) \mid 0 \le x \le 1,\; 0 \le y \le 1 - x\}\) を求めよ。

    解答

    \(x\) で外側を積分する。

    \[ \int_0^1 \left( \int_0^{1-x} (x + y)\, dy \right) dx = \int_0^1 \left[ xy + \frac{y^2}{2} \right]_0^{1-x} dx \]
    \[ = \int_0^1 \left[ x(1 - x) + \frac{(1 - x)^2}{2} \right] dx \]
    \[ = \int_0^1 \left[ x - x^2 + \frac{1}{2} - x + \frac{x^2}{2} \right] dx = \int_0^1 \left( \frac{1}{2} - \frac{x^2}{2} \right) dx \]
    \[ = \left[ \frac{x}{2} - \frac{x^3}{6} \right]_0^1 = \frac{1}{2} - \frac{1}{6} = \frac{1}{3} \]

    演習3（教科書 p.232）

    \(\displaystyle\iint_D \sqrt{x}\, dx\, dy\), \(D = \{(x, y) \mid 0 \le x \le 1,\; 0 \le y \le \tfrac{1}{2},\; y^2 \le x\}\) を求めよ。

    解答

    \(y\) で外側を積分する。\(x\) の範囲は \(y^2 \le x \le 1\)。

    \[ \int_0^{1/2} \left( \int_{y^2}^1 \sqrt{x}\, dx \right) dy = \int_0^{1/2} \left[ \frac{2}{3} x^{3/2} \right]_{y^2}^1 dy \]
    \[ = \int_0^{1/2} \frac{2}{3} \left( 1 - y^3 \right) dy = \frac{2}{3} \left[ y - \frac{y^4}{4} \right]_0^{1/2} \]
    \[ = \frac{2}{3} \left( \frac{1}{2} - \frac{1}{64} \right) = \frac{2}{3} \cdot \frac{31}{64} = \frac{31}{96} \]
