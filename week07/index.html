    第7回：陰関数の微分

    7.1 陰関数と陽関数

    \(y = f(x)\) のように \(y\) が \(x\) の式として明示的に表されている関数を陽関数（explicit function）という。

    一方、

    \[ F(x, y) = 0 \]

    と書けるが、\(y = f(x)\) の形に明示的に書けない関数を陰関数（implicit function）という。

    例

    \(F(x, y) = y - x\sin y = 0\)

    \(x\) を決めると \(y\) の値が決まる（一意とは限らない）。つまり \(y\) は \(x\) の関数であり

    \[ F(x,\, y(x)) = 0 \]

    と書ける。

    7.2 陰関数のグラフ

    方程式 グラフ
    \(F(x, y) = 0\) 曲線（2次元）
    \(F(x, y, z) = 0\) 曲面（3次元）
    \(F(x, y, z) = 0\), \(G(x, y, z) = 0\) 3次元曲線（2曲面の交線）

    7.3 \(F(x, y) = 0\) の微分

    \(F(x,\, y(x)) = 0\) の両辺を \(x\) で微分すると、\(y\) は \(x\) の関数であるから連鎖律より

    \[ \frac{\mathrm{d}F}{\mathrm{d}x} = \frac{\partial F}{\partial x} + \frac{\partial F}{\partial y}\frac{\mathrm{d}y}{\mathrm{d}x} = 0 \]

    陰関数定理（2変数）

    \[ \frac{\mathrm{d}y}{\mathrm{d}x} = -\frac{F_x}{F_y} \]

    例（教科書 p.205）

    \(x^2 + 3xy - 4y^2 - 5 = 0\) のとき \(\dfrac{\mathrm{d}y}{\mathrm{d}x}\) を求めよ。

    解

    方法1：直接微分する

    両辺を \(x\) で微分する（\(y\) は \(x\) の関数であることに注意）。

    \[ 2x + 3y + 3x\frac{\mathrm{d}y}{\mathrm{d}x} - 8y\frac{\mathrm{d}y}{\mathrm{d}x} = 0 \]
    \[ (3x - 8y)\frac{\mathrm{d}y}{\mathrm{d}x} = -(2x + 3y) \]
    \[ \therefore\quad \frac{\mathrm{d}y}{\mathrm{d}x} = -\frac{2x + 3y}{3x - 8y} \qquad (3x \neq 8y) \]

    方法2：陰関数定理を使う

    \(F(x, y) = x^2 + 3xy - 4y^2 - 5\) とおくと

    \[ F_x = 2x + 3y, \qquad F_y = 3x - 8y \]
    \[ \therefore\quad \frac{\mathrm{d}y}{\mathrm{d}x} = -\frac{F_x}{F_y} = -\frac{2x + 3y}{3x - 8y} \]

    7.4 \(F(x, y, z) = 0\) の微分

    \(z = z(x, y)\) とみなし、\(u = F(x,\, y,\, z(x, y)) = 0\) の両辺を偏微分する。

    \(x\) で偏微分すると

    \[ \frac{\partial u}{\partial x} = \frac{\partial F}{\partial x} + \frac{\partial F}{\partial z}\frac{\partial z}{\partial x} = 0 \]

    \(y\) で偏微分すると

    \[ \frac{\partial u}{\partial y} = \frac{\partial F}{\partial y} + \frac{\partial F}{\partial z}\frac{\partial z}{\partial y} = 0 \]

    陰関数定理（3変数）

    \[ \frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \qquad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} \]

    例

    \(x^3 - xyz^2 + z^3 + 3 = 0\) のとき \(\dfrac{\partial z}{\partial x}\), \(\dfrac{\partial z}{\partial y}\) を求めよ。

    解

    \(F(x, y, z) = x^3 - xyz^2 + z^3 + 3\) とおくと

    \[ F_x = 3x^2 - yz^2, \qquad F_y = -xz^2, \qquad F_z = -2xyz + 3z^2 \]
    \[ \frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{3x^2 - yz^2}{-2xyz + 3z^2} \]
    \[ \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = -\frac{-xz^2}{-2xyz + 3z^2} \]

    7.5 連立方程式の場合

    \(F(x, y, z) = 0\), \(G(x, y, z) = 0\) が同時に成り立つとき、2つの曲面の交線として3次元曲線が定まる。

    $x$ を決めると $y$, $z$ の値が決まる。

    \(x\) を決めると \(y\), \(z\) の値が決まるので、\(y = y(x)\), \(z = z(x)\) とみなせる。

    \(u = F(x, y, z) = 0\), \(v = G(x, y, z) = 0\)として、それぞれの式を \(x\) で微分すると

    \[ \begin{cases} \displaystyle \frac{\mathrm{d}u}{\mathrm{d}x} = \frac{\partial F}{\partial x} + \frac{\partial F}{\partial y}\frac{\mathrm{d}y}{\mathrm{d}x} + \frac{\partial F}{\partial z}\frac{\mathrm{d}z}{\mathrm{d}x} = 0 \\[12pt] \displaystyle \frac{\mathrm{d}v}{\mathrm{d}x} = \frac{\partial G}{\partial x} + \frac{\partial G}{\partial y}\frac{\mathrm{d}y}{\mathrm{d}x} + \frac{\partial G}{\partial z}\frac{\mathrm{d}z}{\mathrm{d}x} = 0 \end{cases} \]

    これは \(\dfrac{\mathrm{d}y}{\mathrm{d}x}\), \(\dfrac{\mathrm{d}z}{\mathrm{d}x}\) に関する連立1次方程式であるから、解くことで \(\dfrac{\mathrm{d}y}{\mathrm{d}x}\), \(\dfrac{\mathrm{d}z}{\mathrm{d}x}\) が求められる。

    演習問題

    \(F(x, y) = 0\) の微分

    演習1（教科書 p.205）

    \(x - y^2 = 0\) のとき \(\dfrac{\mathrm{d}y}{\mathrm{d}x}\) を求めよ。

    解答

    \(F(x, y) = x - y^2\) とおくと

    \[ F_x = 1, \qquad F_y = -2y \]
    \[ \frac{\mathrm{d}y}{\mathrm{d}x} = -\frac{F_x}{F_y} = -\frac{1}{-2y} = \frac{1}{2y} \]

    演習2（教科書 p.205）

    \(x^2 + 3xy + y^2 - 6 = 0\) のとき \(\dfrac{\mathrm{d}y}{\mathrm{d}x}\) を求めよ。

    解答

    \(F(x, y) = x^2 + 3xy + y^2 - 6\) とおくと

    \[ F_x = 2x + 3y, \qquad F_y = 3x + 2y \]
    \[ \frac{\mathrm{d}y}{\mathrm{d}x} = -\frac{2x + 3y}{3x + 2y} \]

    演習3（教科書 p.205）

    \(x^3 - 3xy + y^3 = 0\) のとき \(\dfrac{\mathrm{d}y}{\mathrm{d}x}\) を求めよ。

    解答

    \(F(x, y) = x^3 - 3xy + y^3\) とおくと

    \[ F_x = 3x^2 - 3y, \qquad F_y = -3x + 3y^2 \]
    \[ \frac{\mathrm{d}y}{\mathrm{d}x} = -\frac{3x^2 - 3y}{-3x + 3y^2} = \frac{x^2 - y}{x - y^2} \]

    \(F(x, y, z) = 0\) の微分

    演習4

    \(z^2 + xy^3 = \dfrac{xz}{y}\) のとき \(\dfrac{\partial z}{\partial x}\), \(\dfrac{\partial z}{\partial y}\) を求めよ。

    解答

    \(F(x, y, z) = z^2 + xy^3 - \dfrac{xz}{y}\) とおくと

    \[ F_x = y^3 - \frac{z}{y}, \qquad F_y = 3xy^2 + \frac{xz}{y^2}, \qquad F_z = 2z - \frac{x}{y} \]

    公式を使うと

    \[ \frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{y^3 - \dfrac{z}{y}}{2z - \dfrac{x}{y}} \]
    \[ \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = -\frac{3xy^2 + \dfrac{xz}{y^2}}{2z - \dfrac{x}{y}} \]

    Tip

    公式を使わなくても、\(z^2 + xy^3 = \dfrac{xz}{y}\) の両辺を直接 \(x\) や \(y\) で偏微分しても同じ結果が得られる。

    連立方程式の微分

    演習5

    連立方程式

    \[ \begin{cases} \sin(x + y) + \sin(y + z) + z = 0 \\ \cos(x + y) + \cos(y + z) + y - 2 = 0 \end{cases} \]

    の \((x, y, z) = (0, 0, 0)\) における \(\dfrac{\mathrm{d}x}{\mathrm{d}z}\), \(\dfrac{\mathrm{d}y}{\mathrm{d}z}\) を求めよ。

    解答

    \(x = x(z)\), \(y = y(z)\) とみなし、\(F\), \(G\) をそれぞれ \(z\) で微分すると

    \[ \begin{cases} \displaystyle 0 = \frac{\partial F}{\partial x}\frac{\mathrm{d}x}{\mathrm{d}z} + \frac{\partial F}{\partial y}\frac{\mathrm{d}y}{\mathrm{d}z} + \frac{\partial F}{\partial z} \\[12pt] \displaystyle 0 = \frac{\partial G}{\partial x}\frac{\mathrm{d}x}{\mathrm{d}z} + \frac{\partial G}{\partial y}\frac{\mathrm{d}y}{\mathrm{d}z} + \frac{\partial G}{\partial z} \end{cases} \]

    各偏導関数を \((0, 0, 0)\) で評価する。

    \(F = \sin(x + y) + \sin(y + z) + z\) より

    \[ F_x(0,0,0) = \cos(x + y)\big|_{(0,0,0)} = 1 \]
    \[ F_y(0,0,0) = \bigl[\cos(x + y) + \cos(y + z)\bigr]\big|_{(0,0,0)} = 2 \]
    \[ F_z(0,0,0) = \bigl[\cos(y + z) + 1\bigr]\big|_{(0,0,0)} = 2 \]

    \(G = \cos(x + y) + \cos(y + z) + y - 2\) より

    \[ G_x(0,0,0) = -\sin(x + y)\big|_{(0,0,0)} = 0 \]
    \[ G_y(0,0,0) = \bigl[-\sin(x + y) - \sin(y + z) + 1\bigr]\big|_{(0,0,0)} = 1 \]
    \[ G_z(0,0,0) = -\sin(y + z)\big|_{(0,0,0)} = 0 \]

    代入すると

    \[ \begin{cases} 0 = 1 \cdot \dfrac{\mathrm{d}x}{\mathrm{d}z} + 2 \cdot \dfrac{\mathrm{d}y}{\mathrm{d}z} + 2 \\[8pt] 0 = 0 \cdot \dfrac{\mathrm{d}x}{\mathrm{d}z} + 1 \cdot \dfrac{\mathrm{d}y}{\mathrm{d}z} + 0 \end{cases} \]

    第2式より \(\dfrac{\mathrm{d}y}{\mathrm{d}z} = 0\)。第1式に代入して

    \[ \therefore\quad \frac{\mathrm{d}x}{\mathrm{d}z} = -2, \qquad \frac{\mathrm{d}y}{\mathrm{d}z} = 0 \]
