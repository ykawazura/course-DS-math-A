    第3回：全微分

    3.1 接平面

    1変数の微分の復習

    1変数関数 \(y = f(x)\) の微分係数を考える。

    図1：1変数関数の接線近似。接線からの誤差 $\varepsilon$ は $\Delta x \to 0$ で $0$ に収束する。
    \[ f(a + \Delta x) = f'(a)\,\Delta x + f(a) + \varepsilon \]

    すなわち

    \[ f(a + \Delta x) - f(a) = f'(a)\,\Delta x + \varepsilon \]

    であり、\(\varepsilon \to 0\)（\(\Delta x \to 0\)）である。

    このとき \(f(a + \Delta x) - f(a)\) を \(\mathrm{d}f\)、\(\Delta x\) を \(\mathrm{d}x\) とおくと

    \[ \mathrm{d}f = f'(a)\,\mathrm{d}x \tag{$\ast$} \]

    従って \(f'(a) = \dfrac{\mathrm{d}f}{\mathrm{d}x}\) である。

    2変数への拡張

    これを2変数の場合に拡張する。

    \[ f(a + \Delta x,\, b + \Delta y) - f(a, b) = \ ? \]
    図2：曲面 $z = f(x, y)$ と接平面。接平面からの誤差 $\varepsilon$ は $(\Delta x, \Delta y) \to (0, 0)$ で $0$ に収束する。

    1変数の場合の類推から

    \[ f(a + \Delta x,\, b + \Delta y) - f(a, b) = f_x(a, b)\,\Delta x + f_y(a, b)\,\Delta y + \varepsilon \]

    と書ける（\(\varepsilon \to 0\) は \(\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0\) のとき）。

    接平面の方程式

    点 \((a, b)\) を通り、曲面 \(z = f(x, y)\) に接する平面（接平面）の方程式は

    \[ \boxed{z - f(a, b) = \frac{\partial f}{\partial x}(a, b)\,(x - a) + \frac{\partial f}{\partial y}(a, b)\,(y - b)} \]

    とする。

    なぜこうなるか？

    点 \((a, b, c)\) を通る平面の方程式は

    \[ p(x - a) + q(y - b) + r(z - c) = 0 \]

    で書ける。求めたい接平面は \((a, b, f(a,b))\) を通り、曲面 \(z = f(x, y)\) に接触する平面である。したがって

    \[ p(x - a) + q(y - b) + r(z - f(a,b)) = 0 \]

    \(y = b\) で切った断面を考える

    図3：$y = b$ で切った断面。$xz$ 平面上で点 $(a, f(a,b))$ を通り傾き $f_x(a,b)$ の直線になる。

    接平面を \(y = b\) で切ると

    \[ p(x - a) + r(z - f(a, b)) = 0 \]

    これは \(xz\) 平面上で点 \((x, z) = (a,\, f(a,b))\) を通り、傾き \(f_x(a, b)\) の直線

    \[ z = f_x(a, b)(x - a) + f(a, b) \]

    であるので、\(p = f_x(a, b)\), \(r = -1\) とする。

    \(x = a\) で切った断面を考える

    図4：$x = a$ で切った断面。$yz$ 平面上で点 $(b, f(a,b))$ を通り傾き $f_y(a,b)$ の直線になる。

    同様に、接平面を \(x = a\) で切ると

    \[ q(y - b) - (z - f(a, b)) = 0 \]

    これは \(yz\) 平面上で点 \((y, z) = (b,\, f(a,b))\) を通り、傾き \(f_y(a, b)\) の直線

    \[ z = f_y(a, b)(y - b) + f(a, b) \]

    であるので、\(q = f_y(a, b)\) とする。

    以上より、接平面の方程式

    \[ f_x(a, b)(x - a) + f_y(a, b)(y - b) - (z - f(a, b)) = 0 \]

    すなわち

    \[ z - f(a, b) = f_x(a, b)(x - a) + f_y(a, b)(y - b) \]

    が得られる。

    3.2 全微分

    \[ f(a + \Delta x,\, b + \Delta y) - f(a, b) = f_x(a, b)\,\Delta x + f_y(a, b)\,\Delta y + \varepsilon \]

    において \(\varepsilon \to 0\)（\(\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0\)）とする。

    このとき \(\Delta x\) を \(\mathrm{d}x\), \(\Delta y\) を \(\mathrm{d}y\), \(f(a + \Delta x, b + \Delta y) - f(a, b)\) を \(\mathrm{d}f\) とおくと

    \[ \mathrm{d}f = f_x(a, b)\,\mathrm{d}x + f_y(a, b)\,\mathrm{d}y \]

    とする。

    全微分可能の定義

    \[ \lim_{\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0} \frac{f(a + \Delta x,\, b + \Delta y) - f(a, b) - f_x(a, b)\,\Delta x - f_y(a, b)\,\Delta y}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} = 0 \]

    が成り立つとき、\(f(x, y)\) は \((a, b)\) において全微分可能であるといい、

    \[ df = \frac{\partial f}{\partial x}(x, y)\,\mathrm{d}x + \frac{\partial f}{\partial y}(x, y)\,\mathrm{d}y \]

    を \(f(x, y)\) の全微分という。

    例題

    例題 1

    \(f(x, y) = \sqrt{1 - x^2 - y^2}\) は点 \((0, 0)\) で全微分可能か？

    解
    \[ \left.\frac{\partial f}{\partial x}\right|_{(x,y)=(0,0)} = \left.\frac{-x}{\sqrt{1 - x^2 - y^2}}\right|_{(x,y)=(0,0)} = 0 \]
    \[ \left.\frac{\partial f}{\partial y}\right|_{(x,y)=(0,0)} = \left.\frac{-y}{\sqrt{1 - x^2 - y^2}}\right|_{(x,y)=(0,0)} = 0 \]

    全微分可能性の条件を確認する：

    \[ \lim_{\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0} \frac{f(\Delta x,\, \Delta y) - f(0, 0) - 0 \cdot \Delta x - 0 \cdot \Delta y}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} \]
    \[ = \lim_{\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0} \frac{\sqrt{1 - (\Delta x)^2 - (\Delta y)^2} - 1}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} \]

    \(\Delta x = r\cos\theta\), \(\Delta y = r\sin\theta\) とおくと

    \[ = \lim_{r \to 0} \frac{\sqrt{1 - r^2} - 1}{r} = \lim_{r \to 0} \frac{-r^2}{r(\sqrt{1 - r^2} + 1)} = 0 \]

    したがって全微分可能である。\(f(0, 0) = 1\), \(f_x(0, 0) = f_y(0, 0) = 0\) より、\((0, 0)\) における全微分は

    \[ df = 0 \cdot \mathrm{d}x + 0 \cdot \mathrm{d}y = 0 \]

    例題 2

    \(f(x, y) = x^2 + y + 1\) は \((0, 0)\) で全微分可能か？

    解
    \[ \left.\frac{\partial f}{\partial x}\right|_{(x,y)=(0,0)} = 2x\Big|_{(x,y)=(0,0)} = 0, \qquad \left.\frac{\partial f}{\partial y}\right|_{(x,y)=(0,0)} = 1 \]

    全微分可能性の条件を確認する：

    \[ \lim_{\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0} \frac{f(\Delta x,\, \Delta y) - f(0, 0) - 0 \cdot \Delta x - 1 \cdot \Delta y}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} \]
    \[ = \lim_{\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0} \frac{(\Delta x)^2 + \Delta y + 1 - 1 - \Delta y}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} = \lim_{\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0} \frac{(\Delta x)^2}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} \]

    \(\Delta x = r\cos\theta\), \(\Delta y = r\sin\theta\) とおくと

    \[ = \lim_{r \to 0} \frac{r^2 \cos^2\theta}{r} = \lim_{r \to 0} r\cos^2\theta = 0 \]

    したがって全微分は存在する。

    \[ \mathrm{d}f = \mathrm{d}y \]

    例題 3

    \[ f(x, y) = \begin{cases} \dfrac{2xy}{x^2 + y^2} & (x, y) \neq (0, 0) \\[8pt] 0 & (x, y) = (0, 0) \end{cases} \]

    は点 \((0, 0)\) において全微分可能か？

    解
    \[ f_x(0, 0) = 0, \qquad f_y(0, 0) = 0 \]

    全微分可能性の条件を確認する：

    \[ \lim_{\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0} \frac{1}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} \cdot \frac{2\,\Delta x\,\Delta y}{(\Delta x)^2 + (\Delta y)^2} \]

    \(\Delta x = r\cos\theta\), \(\Delta y = r\sin\theta\) とおくと

    \[ = \lim_{r \to 0} \frac{1}{r} \cdot \frac{2r^2 \sin\theta\cos\theta}{r^2} = \lim_{r \to 0} \frac{2\sin\theta\cos\theta}{r} \]

    これは \(0\) に収束しないので、全微分不可能である。

    全微分可能と連続の関係

    全微分可能であれば連続である。

    （逆は成り立たない。また、前回見たように偏微分可能でも連続とは限らない。）

    3.3 法線ベクトルと法線方程式

    2次元の場合

    まず2次元の場合を考える。

    図5：直線$AP$ と垂直ベクトル $\vec{n}$ 。

    直線上に点 \(A\colon(a, b)\) と点 \(P\colon(x, y)\) をとる。\(A\) を通り、この直線の垂直なベクトルを \(\vec{n} = \begin{pmatrix} p \\ q \end{pmatrix}\) とおく。

    直交条件は

    \[ \overrightarrow{AP} \cdot \vec{n} = \begin{pmatrix} x - a \\ y - b \end{pmatrix} \cdot \begin{pmatrix} p \\ q \end{pmatrix} = p(x - a) + q(y - b) = 0 \]

    法線ベクトル

    この直線が曲線 \(y = f(x)\) の接線であるとする。接線の方程式は

    \[ y = f'(a)(x - a) + f(a) \]

    であるので \(p = f'(a)\), \(q = -1\) となる。したがって、曲線 \(y = f(x)\) の点 \((a, f(a))\) における法線ベクトルは

    \[ \vec{n} = \begin{pmatrix} f'(a) \\ -1 \end{pmatrix} \]
    図6：曲線 $y = f(x)$ の点 $A\colon(a, f(a))$ における接線、法線ベクトル $\vec{n}$。

    法線の方程式

    次に \((a, f(a))\) を通り \(\vec{n}\) に平行な直線（法線）の式を導く。

    点 \(Q\colon(x, y)\) を法線上にとると、媒介変数 \(t\) を用いて

    \[ \overrightarrow{OQ} = \overrightarrow{OA} + t\vec{n} \]
    \[ \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} a \\ f(a) \end{pmatrix} + t \begin{pmatrix} f'(a) \\ -1 \end{pmatrix} = \begin{pmatrix} a + t\,f'(a) \\ f(a) - t \end{pmatrix} \]

    \(f'(a) \neq 0\) のとき、\(t\) を消去すると

    \[ \frac{x - a}{f'(a)} = -(y - f(a)) \]
    図7：曲線 $y = f(x)$ の点 $A\colon(a, f(a))$ における法線。

    3次元の場合

    上と同じ話を3次元に拡張する。

    図8：曲面 $z = f(x, y)$ の点 $A$ における接平面、法線ベクトル $\vec{n}$、法線。

    点 \(A\colon(a, b, f(a,b))\) で曲面 \(z = f(x, y)\) に接する平面は

    \[ z - f(a, b) = f_x(a, b)(x - a) + f_y(a, b)(y - b) \]

    この平面上の点を \(P\colon(x, y, z)\)、\(A\) を通る法線ベクトルを \(\vec{n} = \begin{pmatrix} p \\ q \\ r \end{pmatrix}\) とすると、直交条件は

    \[ \overrightarrow{AP} \cdot \vec{n} = \begin{pmatrix} x - a \\ y - b \\ z - f(a,b) \end{pmatrix} \cdot \begin{pmatrix} p \\ q \\ r \end{pmatrix} = p(x - a) + q(y - b) + r(z - f(a,b)) = 0 \]

    一方、接平面の式は \(z - f(a,b) = f_x(a,b)(x-a) + f_y(a,b)(y-b)\) であるので

    \[ \vec{n} = \begin{pmatrix} f_x(a, b) \\ f_y(a, b) \\ -1 \end{pmatrix} \]

    とする。

    法線の方程式

    法線の式は媒介変数 \(t\) を用いて

    \[ \overrightarrow{OQ} = \overrightarrow{OA} + t\vec{n} \quad \Longleftrightarrow \quad \begin{pmatrix} x \\ y \\ z \end{pmatrix} = \begin{pmatrix} a \\ b \\ f(a,b) \end{pmatrix} + t \begin{pmatrix} f_x(a, b) \\ f_y(a, b) \\ -1 \end{pmatrix} \]

    であり、\(f_x(a, b) \neq 0\) かつ \(f_y(a, b) \neq 0\) のとき

    \[ \boxed{\frac{x - a}{f_x(a, b)} = \frac{y - b}{f_y(a, b)} = -(z - f(a, b))} \]

    とする。

    例題

    例題 1

    \(f(x, y) = x^2 - y^2\) の点 \((-2, 1)\) における接平面の方程式と法線方程式を求めよ。

    解
    \[ f_x(-2, 1) = 2x\Big|_{(x,y)=(-2,1)} = -4, \qquad f_y(-2, 1) = -2y\Big|_{(x,y)=(-2,1)} = -2 \]
    \[ f(-2, 1) = 4 - 1 = 3 \]

    接平面：

    \[ z - 3 = -4(x + 2) - 2(y - 1) \]
    \[ \Longleftrightarrow \quad z = -4x - 2y - 3 \]

    法線：

    \[ \frac{x + 2}{-4} = \frac{y - 1}{-2} = -(z - 3) \]

    例題 2

    \(f(x, y) = \ln(x^2 + y^2)\) の点 \((1, -2)\) における接平面の方程式と法線方程式を求めよ。

    解
    \[ f_x(1, -2) = \frac{2x}{x^2 + y^2}\bigg|_{(x,y)=(1,-2)} = \frac{2}{5}, \qquad f_y(1, -2) = \frac{2y}{x^2 + y^2}\bigg|_{(x,y)=(1,-2)} = -\frac{4}{5} \]
    \[ f(1, -2) = \ln 5 \]

    接平面：

    \[ z - \ln 5 = \frac{2}{5}(x - 1) - \frac{4}{5}(y + 2) \]
    \[ \Longleftrightarrow \quad z = \frac{2}{5}x - \frac{4}{5}y - 2 + \ln 5 \]

    法線：

    \[ \frac{x - 1}{2/5} = \frac{y + 2}{-4/5} = -(z - \ln 5) \]
    \[ \Longleftrightarrow \quad \frac{x - 1}{2} = \frac{y + 2}{-4} = \frac{-(z - \ln 5)}{5} \]
