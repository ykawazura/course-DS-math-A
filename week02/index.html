    第2回：偏微分

    2変数関数の極限

    1変数の場合の復習

    \(x\) が \(a\) と異なる値をとりながら \(a\) に近づくとき、\(f(x)\) の値が \(A\) に近づくならば

    \[ \lim_{x \to a} f(x) = A \]

    と書く。

    図1：1変数関数の極限。左側極限と右側極限が一致するとき極限が存在する。

    極限の存在条件：右側極限と左側極限が一致すること。

    \[ \lim_{x \to +a} f(x) = \lim_{x \to -a} f(x) = A \]

    2変数の場合

    2変数の場合、\(f(x, y)\) において \((x, y)\) のある点 \((a, b)\) への近づけ方が無数にある。

    図2：原点への近づけ方は無限にある。直線 $y = 0$、$y = x$、$y = -x$ など。

    注意

    極限を考える点は定義域 \(D\) に含まれる必要はない。

    例

    \(\displaystyle f(x, y) = \frac{2xy}{x^2 + y^2}\), \((x, y) \neq (0, 0)\)

    \((x, y) \to (0, 0)\) での極限はどうなるか？

    ① 直線 \(y = x\) に沿って近づける

    \[ f(t, t) = \frac{2t^2}{t^2 + t^2} = 1 \quad \therefore\ \lim_{t \to 0} f(t, t) = 1 \]

    ② \(x\) 軸に沿って近づける (\(y = 0\))

    \[ f(t, 0) = \frac{0}{t^2} = 0 \quad \therefore\ \lim_{t \to 0} f(t, 0) = 0 \]

    経路によって極限値が異なるので、極限は存在しない。

    極座標による判定法

    \(f(x, y)\) が点 \((a, b)\) で極限をもつためには \((x, y) \to (a, b)\) の近づけ方によらず \(f(x, y)\) がある値 \(A\) に近づく必要がある。

    図3：極座標で半径 $r$ を $0$ に縮めながら近づける。

    \(\sqrt{(x-a)^2 + (y-b)^2} = r\) とすると、\(x = a + r\cos\theta\), \(y = b + r\sin\theta\) である。

    \((x, y) \to (a, b)\) の近づけ方によらず \(f(x, y)\) がある値 \(A\) に近づくということは、

    \[ |f(a + r\cos\theta,\, b + r\sin\theta) - A| \to 0 \quad (r \to 0) \]

    が \(\theta\) に依存しないことである。

    このとき、\(\displaystyle\lim_{(x,y) \to (a,b)} f(x,y) = A\) と書く。

    例題

    例題 1

    \(\displaystyle\lim_{(x,y) \to (0,0)} \frac{2x - y + 5}{1 + x^2 + y^2}\)

    解：分母・分子ともに連続なので、そのまま代入できる。

    \[ = \frac{0 - 0 + 5}{1 + 0 + 0} = 5 \]

    例題 2

    \(\displaystyle\lim_{(x,y) \to (0,0)} \frac{x}{x^2 + y^2}\)

    解：\(x\) 軸に沿って近づける（\(y = 0\)）と

    \[ f(x, 0) = \frac{1}{x} \]

    \(\displaystyle\lim_{x \to +0} \frac{1}{x} = +\infty\), \(\displaystyle\lim_{x \to -0} \frac{1}{x} = -\infty\) であるから、極限は存在しない。

    例題 3

    \(\displaystyle\lim_{(x,y) \to (0,0)} \frac{x^3}{x^2 + y^2}\)

    解：

    • \(x\) 軸に沿う：\(f(x, 0) = x \to 0\)
    • \(y\) 軸に沿う：\(f(0, y) = 0\)
    • \(y = x\) に沿う：\(f(x, x) = x/2 \to 0\)

    すべての経路で \(0\) だが、これだけでは不十分。極座標で確認する。

    \[ f(r\cos\theta,\, r\sin\theta) = \frac{r^3 \cos^3\theta}{r^2(\cos^2\theta + \sin^2\theta)} = r\cos^3\theta \]

    \(|\cos^3\theta| \leq 1\) なのではさみうちの原理を使って $$ |f(r\cos\theta,\, r\sin\theta) - 0| \leq |r| \to 0 \quad (r \to 0) $$

    \[ \therefore\ \lim_{(x,y) \to (0,0)} \frac{x^3}{x^2 + y^2} = 0 \]

    例題 4

    \(\displaystyle\lim_{(x,y) \to (0,0)} \frac{xy}{\sqrt{x^2 + 3y^2}}\)

    解：極座標で

    \[ f(r\cos\theta,\, r\sin\theta) = \frac{r^2\cos\theta\sin\theta}{r\sqrt{\cos^2\theta + 3\sin^2\theta}} = \frac{r\cos\theta\sin\theta}{\sqrt{1 + 2\sin^2\theta}} \]
    \[ |f(r\cos\theta,\, r\sin\theta) - 0| \leq |r| \to 0 \quad (r \to 0) \]
    \[ \therefore\ \lim_{(x,y) \to (0,0)} \frac{xy}{\sqrt{x^2 + 3y^2}} = 0 \]

    例題 5

    \(\displaystyle\lim_{(x,y) \to (0,0)} \frac{\sin(xy)}{x^2 + y^2}\)

    解：

    \[ \frac{\sin(xy)}{x^2 + y^2} = \frac{\sin(xy)}{xy} \cdot \frac{xy}{x^2 + y^2} \]

    第1因子は \(\displaystyle\lim_{t \to 0} \frac{\sin t}{t} = 1\) より \(1\) に近づく。

    第2因子は \(y = x\) に沿うと \(1/2\)、\(y = 0\) に沿うと \(0\) であり、極限なし。

    \(\therefore\) 極限は存在しない。

    2.1 極限の性質

    \(D \subset \mathbb{R}^n\), \(f \colon D \to \mathbb{R}\), \(g \colon D \to \mathbb{R}\) とする。

    \(A \in D\) に対し \(\displaystyle\lim_{P \to A} f(P) = \alpha\) かつ \(\displaystyle\lim_{P \to A} g(P) = \beta\) のとき：

    1. \(c, d \in \mathbb{R}\) に対し \(\displaystyle\lim_{P \to A} \left[c\,f(P) + d\,g(P)\right] = c\alpha + d\beta\)

    2. \(\displaystyle\lim_{P \to A} f(P)\,g(P) = \alpha\beta\)

    3. \(g(P) \neq 0\) (\(P \in D\)) かつ \(\beta \neq 0\) のとき \(\displaystyle\lim_{P \to A} \frac{f(P)}{g(P)} = \frac{\alpha}{\beta}\)

    2.2 連続関数

    \(f\) が \(D\) の点 \((a_1, \ldots, a_n)\) で連続であるとは

    \[ \lim_{(x_1, \ldots, x_n) \to (a_1, \ldots, a_n)} f(x_1, \ldots, x_n) = f(a_1, \ldots, a_n) \]

    が成り立つことである。

    図4：連続関数と不連続関数の例。

    例題

    次の関数は \((0, 0)\) で連続か？

    \[ f(x, y) = \begin{cases} \dfrac{x^4 - 3x^2y^2}{2x^2 + y^2} & (x, y) \neq (0, 0) \\[8pt] 0 & (x, y) = (0, 0) \end{cases} \]

    解：極座標で

    \[ f(r\cos\theta,\, r\sin\theta) = r^2 \cdot \frac{\cos^2\theta(\cos^2\theta - 3\sin^2\theta)}{2\cos^2\theta + \sin^2\theta} \]
    \[ |f(r\cos\theta,\, r\sin\theta) - 0| \leq r^2 \to 0 \quad (r \to 0) = f(0, 0) \]

    \(\therefore\) \((0, 0)\) で連続である。

    連続関数の演算

    関数 \(f\) と \(g\) が \(D\) の点 \(A\) で連続ならば

    • \(cf + dg\) も \(A\) で連続
    • \(fg\) も \(A\) で連続
    • \(f/g\) も \(A\) で連続（\(g(P) \neq 0\) のとき）

    2.3 偏微分（Partial derivative）

    1変数の微分の復習

    \(y = f(x)\) の微分（または導関数）は

    \[ f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h} \]

    \(f'(x)\) は \(\dfrac{\mathrm{d}f(x)}{\mathrm{d}x}\) とも書く。

    \(x = a\) における \(f(x)\) の導関数の値 \(f'(a)\) を \(f(x)\) の \(x = a\) における微分係数という。

    これは \(y = f(x)\) の \(x = a\) における接線の傾きを表す。

    2変数への拡張

    \(z = f(x, y)\) は曲面なので、微分係数を求めることは \((x, y) = (a, b)\) における接平面の傾きを求めることである。

    図5：曲面 $z = f(x, y)$ と点 $(a, b)$ における接平面。

    平面の方程式は \(z = \alpha x + \beta y\) の形なので、傾きは2つある。

    \(x\) に関する偏微分

    曲面 \(z = f(x, y)\) を \(y = b\) で切った断面 \(z = f(x, b)\) を考える。

    図6：$y = b$ で切った断面と $x$ 方向の偏微分。この傾きが $\left.\dfrac{\mathrm{d}f(x, b)}{\mathrm{d}x}\right|_{x=a}$ である。

    \(f(x, y)\) が点 \((a, b)\) において \(x\) に関して偏微分可能とは、\(F(x) = f(x, b)\) が \(x = a\) で微分可能、すなわち

    \[ F'(a) = \lim_{h \to 0} \frac{F(a + h) - F(a)}{h} = \lim_{h \to 0} \frac{f(a + h,\, b) - f(a,\, b)}{h} \]

    が存在することである。この \(F'(a)\) を、\(f\) の \((a, b)\) における \(x\) に関する偏微分係数 といい、

    \[ f_x(a, b), \quad \frac{\partial f}{\partial x}(a, b), \quad \left.\frac{\partial f(x, y)}{\partial x}\right|_{(x,y)=(a,b)} \]

    と書く（\(\partial\) はラウンド、デル、パーシャルなどと呼ばれる）。

    \(y\) に関する偏微分

    同様に、\(G(y) = f(a, y)\) として

    図7：$x = a$ で切った断面と $y$ 方向の偏微分。この傾きが $\left.\dfrac{\mathrm{d}f(a, y)}{\mathrm{d}y}\right|_{y=b}$ である。
    \[ G'(b) = \lim_{h \to 0} \frac{G(b + h) - G(b)}{h} = \lim_{h \to 0} \frac{f(a,\, b + h) - f(a,\, b)}{h} \]

    を \(f\) の \((a, b)\) における \(y\) に関する偏微分係数 といい、\(f_y(a, b)\), \(\dfrac{\partial f}{\partial y}(a, b)\) と書く。

    偏導関数

    \[ f_x(x, y) = \frac{\partial f(x, y)}{\partial x} = \lim_{h \to 0} \frac{f(x + h,\, y) - f(x,\, y)}{h} \]

    を \(f(x, y)\) の \(x\) に関する偏導関数 という。

    同様に

    \[ f_y(x, y) = \frac{\partial f(x, y)}{\partial y} = \lim_{h \to 0} \frac{f(x,\, y + h) - f(x,\, y)}{h} \]

    を \(f(x, y)\) の \(y\) に関する偏導関数 という。

    \(f(x, y)\) から \(f_x(x, y)\) を求めることを \(f\) を \(x\) で偏微分する という。

    3変数以上への拡張

    3変数以上の場合も同様に定義される。

    \[ \frac{\partial f(x, y, z)}{\partial x} = \lim_{h \to 0} \frac{f(x + h,\, y,\, z) - f(x,\, y,\, z)}{h} \]

    偏微分の計算方法

    \(\dfrac{\partial f}{\partial x}\) は \(x\) 以外の変数をすべて定数とみなして、\(x\) で1変数と同様に微分すればよい。

    例題 1

    \(f(x, y) = x^2 y^3\)

    解：

    定義に基づく計算：

    \[ \frac{\partial f(x, y)}{\partial x} = \lim_{h \to 0} \frac{(x+h)^2 y^3 - x^2 y^3}{h} = \lim_{h \to 0} \frac{(x^2 + 2xh + h^2)y^3 - x^2 y^3}{h} \]
    \[ = \lim_{h \to 0} (2xy^3 + h y^3) = 2xy^3 \]

    同様に

    \[ \frac{\partial f(x, y)}{\partial y} = \lim_{h \to 0} \frac{x^2(y+h)^3 - x^2 y^3}{h} = \lim_{h \to 0} (3x^2 y^2 + 3x^2 y h + x^2 h^2) = 3x^2 y^2 \]

    \(y\) を定数とみなして微分すれば、直ちに \(\dfrac{\partial f}{\partial x} = 2xy^3\) が得られる。

    例題 2（p.184）

    \(\displaystyle f(x, y) = \frac{x}{y}\)

    解：

    \[ \frac{\partial f}{\partial x} = \frac{1}{y}, \qquad \frac{\partial f}{\partial y} = -\frac{x}{y^2} \]

    例題 3（p.184）

    \(f(x, y) = \sin(2x - y)\)

    解：

    \[ \frac{\partial f}{\partial x} = 2\cos(2x - y), \qquad \frac{\partial f}{\partial y} = -\cos(2x - y) \]

    例題 4（p.185）

    \(f(x, y) = e^x \sin y\)

    解：

    \[ \frac{\partial f}{\partial x} = e^x \sin y, \qquad \frac{\partial f}{\partial y} = e^x \cos y \]

    偏微分可能性と連続性

    1変数のときは「微分可能 \(\Rightarrow\) 連続」であった。

    2変数のときは 「偏微分可能 \(\Rightarrow\) 連続」は成り立たない。

    反例

    \[ f(x, y) = \begin{cases} \dfrac{2xy}{x^2 + y^2} & (x, y) \neq (0, 0) \\[8pt] 0 & (x, y) = (0, 0) \end{cases} \]

    偏微分可能であること：

    \[ f_x(0, 0) = \lim_{h \to 0} \frac{f(h, 0) - f(0, 0)}{h} = \lim_{h \to 0} \frac{\frac{2h \cdot 0}{h^2} - 0}{h} = 0 \]
    \[ f_y(0, 0) = \lim_{h \to 0} \frac{f(0, h) - f(0, 0)}{h} = \lim_{h \to 0} \frac{\frac{2 \cdot 0 \cdot h}{h^2} - 0}{h} = 0 \]

    \(\therefore\) 偏微分可能。

    連続でないこと：\(y = x\) に沿って \((0, 0)\) に近づけると

    \[ f(x, x) = \frac{2x^2}{x^2 + x^2} = 1 \neq 0 = f(0, 0) \]

    \(\therefore\) 連続でない。

    連続であるためには、\((x, y)\) をすべての方向から近づけても \(f(x, y)\) が同じ値に近づく必要がある。一方、偏微分は \(x\) 軸・\(y\) 軸に沿った変化しか見ていない。

    \(\longrightarrow\) 「全微分」 が必要（次回以降で扱う）。

    演習問題

    演習 1

    \(\displaystyle f(x, y) = \frac{x}{x + y}\) の偏導関数を求めよ。

    解答
    \[ f_x(x, y) = \frac{1 \cdot (x + y) - x \cdot 1}{(x + y)^2} = \frac{y}{(x + y)^2} \]
    \[ f_y(x, y) = \frac{- x \cdot 1}{(x + y)^2} = -\frac{x}{(x + y)^2} \]

    演習 2

    \(f(x, y) = \sqrt{x^2 + y^2}\) の偏導関数を求めよ。

    解答
    \[ f_x(x, y) = \frac{x}{\sqrt{x^2 + y^2}}, \qquad f_y(x, y) = \frac{y}{\sqrt{x^2 + y^2}} \]

    演習 3

    \(f(x, y) = e^{ax} \sin(by)\) の偏導関数を求めよ。

    解答
    \[ f_x(x, y) = ae^{ax} \sin(by), \qquad f_y(x, y) = be^{ax} \cos(by) \]

    演習 4

    \(f(x, y) = \ln(x^2 + y^2)\), \((x, y) \neq (0, 0)\) の偏導関数を求めよ。

    解答
    \[ f_x(x, y) = \frac{2x}{x^2 + y^2}, \qquad f_y(x, y) = \frac{2y}{x^2 + y^2} \]

    演習 5

    \(\displaystyle f(x, y) = \tan^{-1}\!\left(\frac{x}{y}\right)\) の偏導関数を求めよ。

    解答

    \(\dfrac{\mathrm{d}}{\mathrm{d}u}\tan^{-1}(u) = \dfrac{1}{1 + u^2}\) を用いる。

    \[ f_x(x, y) = \frac{1}{1 + \left(\frac{x}{y}\right)^2} \cdot \frac{1}{y} = \frac{y}{x^2 + y^2} \]
    \[ f_y(x, y) = \frac{1}{1 + \left(\frac{x}{y}\right)^2} \cdot \left(-\frac{x}{y^2}\right) = -\frac{x}{x^2 + y^2} \]
