    第4回：高階偏導関数

    4.1 高次偏導関数

    1変数の復習

    1変数関数の場合、\(n\) 次導関数は

    \[ \frac{\mathrm{d}^n f}{\mathrm{d}x^n} = f^{(n)}(x) \]

    で定義される。ここでは \(n = 2\)（2次偏導関数）の場合を考える。

    2変数の高次偏導関数

    \(f(x, y)\) について \(f_x(x, y)\), \(f_y(x, y)\) が存在し、さらにそれぞれが微分可能であるとき、以下の4つの2次偏導関数が定義できる。

    \[ f_{xx}(x, y) = \frac{\partial}{\partial x}\!\left(\frac{\partial f}{\partial x}\right) = \frac{\partial^2 f}{\partial x^2}(x, y) \]
    \[ f_{xy}(x, y) = \frac{\partial}{\partial y}\!\left(\frac{\partial f}{\partial x}\right) = \frac{\partial^2 f}{\partial y\,\partial x}(x, y) \]
    \[ f_{yx}(x, y) = \frac{\partial}{\partial x}\!\left(\frac{\partial f}{\partial y}\right) = \frac{\partial^2 f}{\partial x\,\partial y}(x, y) \]
    \[ f_{yy}(x, y) = \frac{\partial}{\partial y}\!\left(\frac{\partial f}{\partial y}\right) = \frac{\partial^2 f}{\partial y^2}(x, y) \]

    例題

    例題 1

    \(f(x, y) = x^3 - 3xy + y^3\) の2次偏導関数をすべて求めよ。

    解
    \[ f_x(x, y) = 3x^2 - 3y, \qquad f_y(x, y) = -3x + 3y^2 \]
    \[ f_{xx}(x, y) = 6x \]
    \[ f_{xy}(x, y) = f_{yx}(x, y) = -3 \]
    \[ f_{yy}(x, y) = 6y \]

    例題 2

    \(f(x, y) = e^x \sin y\) の2次偏導関数をすべて求めよ。

    解
    \[ f_x = e^x \sin y, \qquad f_y = e^x \cos y \]
    \[ f_{xx} = e^x \sin y \]
    \[ f_{xy} = f_{yx} = e^x \cos y \]
    \[ f_{yy} = -e^x \sin y \]

    例題 3

    \(f(x, y) = \ln(x^2 - y^2)\) の2次偏導関数をすべて求めよ。

    解
    \[ f_x = \frac{2x}{x^2 - y^2}, \qquad f_y = \frac{-2y}{x^2 - y^2} \]
    \[ f_{xx} = \frac{2(x^2 - y^2) - 2x \cdot 2x}{(x^2 - y^2)^2} = \frac{-2(x^2 + y^2)}{(x^2 - y^2)^2} \]
    \[ f_{xy} = f_{yx} = \frac{4xy}{(x^2 - y^2)^2} \]
    \[ f_{yy} = \frac{-2(x^2 - y^2) - (-2y)(-2y)}{(x^2 - y^2)^2} = \frac{-2(x^2 + y^2)}{(x^2 - y^2)^2} \]

    4.2 シュワルツの定理

    一般に \(f_{xy}\) と \(f_{yx}\) は同じとは限らない。

    シュワルツの定理

    \(f_x(x, y)\), \(f_y(x, y)\), \(f_{xy}(x, y)\) が存在し、\(f_{xy}\) が連続ならば、\(f_{yx}\) が存在して

    \[ f_{xy}(x, y) = f_{yx}(x, y) \]

    が成り立つ。

    \(f_{xy} \neq f_{yx}\) の反例

    以下の関数は \((0, 0)\) で \(f_{xy} \neq f_{yx}\) となる。

    \[ f(x, y) = \begin{cases} \dfrac{xy(x^2 - y^2)}{x^2 + y^2} & (x, y) \neq (0, 0) \\[8pt] 0 & (x, y) = (0, 0) \end{cases} \]

    \((x, y) \neq (0, 0)\) のとき

    \[ f_x(x, y) = \frac{x^4 y + 4x^2 y^3 - y^5}{(x^2 + y^2)^2}, \qquad f_y(x, y) = \frac{x^5 - 4x^3 y^2 - x y^4}{(x^2 + y^2)^2} \]

    \(x = r\cos\theta\), \(y = r\sin\theta\) とおくと

    \[ |f_x(x, y)| = |r(\cos^4\!\theta\,\sin\theta + 4\cos^2\!\theta\,\sin^3\!\theta - \sin^5\!\theta)| \leq 6r \to 0 \quad (r \to 0) \]
    \[ |f_y(x, y)| = |r(\cos^5\!\theta - 4\cos^3\!\theta\,\sin^2\!\theta - \cos\theta\,\sin^4\!\theta)| \leq 6r \to 0 \quad (r \to 0) \]

    したがって \(f_x(x, y)\) および \(f_y(x, y)\) は \((x, y) = (0, 0)\) で連続である（\(f_x(0,0) = f_y(0,0) = 0\)）。

    \((x, y) \neq (0, 0)\) のとき、\(f_{xy}\) と \(f_{yx}\) を計算すると

    \[ f_{xy}(x, y) = f_{yx}(x, y) = \frac{x^6 + 9x^4 y^2 - 9x^2 y^4 - y^6}{(x^2 + y^2)^3} \]

    \(x = r\cos\theta\), \(y = r\sin\theta\) とおくとこれは \(\cos 2\theta\,(1 + 2\sin^2\!2\theta)\) となり、\(\theta\) に依存して値が変わるので \((0, 0)\) で連続ではない。

    実際に \((0, 0)\) では

    \[ f_{xy}(0, 0) = \lim_{h \to 0}\frac{f_x(0, h) - f_x(0, 0)}{h} = \lim_{h \to 0}\frac{\dfrac{-h^5}{h^4} - 0}{h} = -1 \]
    \[ f_{yx}(0, 0) = \lim_{h \to 0}\frac{f_y(h, 0) - f_y(0, 0)}{h} = \lim_{h \to 0}\frac{\dfrac{h^5}{h^4} - 0}{h} = 1 \]

    となり、\(f_{xy}(0, 0) \neq f_{yx}(0, 0)\) である。

    \(C^n\) 級

    \(C^n\) 級の定義

    1. \(n\) 階までの全ての偏導関数が存在して連続であるとき、\(f(x, y)\) は \(C^n\) 級（\(n\) 階連続微分可能）であるという。
    2. 任意の回数微分可能であるとき、\(C^\infty\) 級であるという。
    3. \(f\) 自身が連続であることを \(C^0\) 級であるという。

    Tip

    \(f(x, y)\) が \(C^n\) 級であれば、\(n\) 階までの偏微分の順序を入れ替えてよい。

    4.3 自然界の具体例

    高次偏導関数は自然界の様々な現象を記述する偏微分方程式に現れる。

    拡散方程式

    \[ \frac{\partial f}{\partial t} = C\,\frac{\partial^2 f}{\partial x^2} \qquad (C\colon \text{定数}) \]

    ここで \(f(x, t)\) は温度、\(t\) は時間、\(x\) は位置を表す。ある初期時刻に局在していた温度分布が、時間とともに広がり平坦になっていく様子を記述する。

    図1：拡散方程式における温度分布の時間発展。初期の鋭い分布 $f(x,\,t{=}0)$ が時間とともに広がり平坦になる。

    線形波動方程式

    \[ \frac{\partial^2 f}{\partial t^2} = C\,\frac{\partial^2 f}{\partial x^2} \qquad (C > 0) \]

    波の伝播を記述する方程式である。弦の振動はこの方程式に従う。

    図2：固定端振動。両端 $x = 0,\, L$ が固定された弦が振動する様子。

    非線形移流方程式（Burgers方程式）

    \[ \frac{\partial f}{\partial t} + f\frac{\partial f}{\partial x} = 0 \]

    非線形項 \(f\,\partial f / \partial x\) により、振幅の大きい部分ほど速く伝播するため、波形が次第に急峻になる（衝撃波の形成）。衝撃波が形成されると \(\partial / \partial x\) と \(\partial / \partial t\) が交換できなくなる。

    図3：非線形移流方程式の解。初期に滑らかな分布が、振幅の大きい部分ほど速く進むことで波形が急峻になる。

    演習問題

    演習 1

    \(f(x, y) = 2x^3 + 3xy^2\) の2次偏導関数をすべて求めよ。

    解答
    \[ f_x(x, y) = 6x^2 + 3y^2, \qquad f_y(x, y) = 6xy \]
    \[ f_{xx}(x, y) = 12x \]
    \[ f_{xy}(x, y) = f_{yx}(x, y) = 6y \]
    \[ f_{yy}(x, y) = 6x \]

    演習 2

    \(f(x, y) = x^3 y + e^{xy}\) の2次偏導関数をすべて求めよ。

    解答
    \[ f_x = 3x^2 y + y\,e^{xy}, \qquad f_y = x^3 + x\,e^{xy} \]
    \[ f_{xx} = 6xy + y^2 e^{xy} \]
    \[ f_{xy} = f_{yx} = 3x^2 + xy\,e^{xy} + e^{xy} \]
    \[ f_{yy} = x^2 e^{xy} \]

    演習 3

    \(f(x, y) = x^2 y + \ln(y^2 - x)\) の2次偏導関数をすべて求めよ。

    解答
    \[ f_x = 2xy - \frac{1}{y^2 - x}, \qquad f_y = x^2 + \frac{2y}{y^2 - x} \]
    \[ f_{xx} = 2y - \frac{1}{(y^2 - x)^2} \]
    \[ f_{xy} = 2x + \frac{2y}{(y^2 - x)^2} \]
    \[ f_{yy} = \frac{2(y^2 - x) - 4y^2}{(y^2 - x)^2} = \frac{-2y^2 - 2x}{(y^2 - x)^2} \]

    これまでの復習

    第1回〜第4回のまとめ

    ① 2変数関数のグラフと定義域

    \(z = f(x, y)\) は3次元空間の曲面を表す。定義域は \(f(x, y)\) が定義できる \((x, y)\) の範囲。定義域に含まれないのは

    1. 分母が \(0\) になる点
    2. ルートの中が負になる点
    3. 対数の中が \(0\) または負になる点

    ② 極限

    \(f(x, y)\) が \((a, b)\) で極限値 \(A\) を持つことを示すには、\(x = a + r\cos\theta\), \(y = b + r\sin\theta\) とおいて

    \[ |f(a + r\cos\theta,\, b + r\sin\theta) - A| \to 0 \quad (r \to 0) \]

    が \(\theta\) によらないことを確認する。もしくは収束値が異なる経路を2つ見つければ極限が存在しないことが証明される。

    ③ 偏微分

    \[ \frac{\partial f}{\partial x}, \qquad \frac{\partial f}{\partial y} \]

    偏微分は、微分する変数以外の変数を定数と思って微分する。

    ④ 全微分

    \[ \lim_{\sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0} \frac{f(a + \Delta x,\, b + \Delta y) - f(a, b) - f_x(a, b)\,\Delta x - f_y(a, b)\,\Delta y}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} = 0 \]

    のとき全微分可能であり、全微分は

    \[ df = \frac{\partial f}{\partial x}\,\mathrm{d}x + \frac{\partial f}{\partial y}\,\mathrm{d}y \]

    ⑤ 接平面

    \[ z - f(a, b) = f_x(a, b)(x - a) + f_y(a, b)(y - b) \]

    ⑥ 法線

    \[ \frac{x - a}{f_x(a, b)} = \frac{y - b}{f_y(a, b)} = -(z - f(a, b)) \]

    ⑦ シュワルツの定理

    \(f_{xy}\) が連続ならば

    \[ f_{xy} = f_{yx} \]
