    第6回：多変数のテイラー展開

    6.1 1変数の復習

    \(f(x)\) が \(a \leq x \leq b\) で連続で、\(a < x < b\) で微分可能ならば

    \[ f(b) = f(a) + f'(a)(b - a) + \frac{f''(a)}{2!}(b - a)^2 + \cdots + \frac{f^{(n-1)}(a)}{(n-1)!}(b - a)^{n-1} + R_n \]

    ここで剰余項は

    \[ R_n = \frac{f^{(n)}(c)}{n!}(b - a)^n \qquad (a < c < b \text{ を満たすある } c \text{ が存在}) \]

    無限階微分可能で \(R_n \to 0\) \((n \to \infty)\) ならば

    \[ f(x) = f(a) + f'(a)(x - a) + \frac{f''(a)}{2!}(x - a)^2 + \cdots \]

    6.2 2変数の場合

    多変数式の「次数」とは

    1変数の場合、べき級数の次数は

    \[ \underbrace{1}_{0\text{次}} + \underbrace{x}_{1\text{次}} + \underbrace{2x^2}_{2\text{次}} + \underbrace{5x^3}_{3\text{次}} + \cdots \]

    のように、各項の \(x\) の指数で決まる。

    2変数の場合は、\(x\) と \(y\) の指数の和で次数が決まる。

    \[ \underbrace{1}_{0\text{次}} + \underbrace{x + 2y}_{1\text{次}} + \underbrace{3x^2 + 4xy + 5y^2}_{2\text{次}} + \cdots \]

    Tip

    \(n\) 次の項は \((ax + by)^n\) の形をしている。

    6.3 2変数のテイラー展開

    \(f(x, y)\) の \((a, b)\) のまわりでのテイラー展開は

    \[ f(x, y) = f(a, b) + f_x(a, b)(x - a) + f_y(a, b)(y - b) \]
    \[ {} + \frac{1}{2!}\bigl[f_{xx}(a, b)(x - a)^2 + 2f_{xy}(a, b)(x - a)(y - b) + f_{yy}(a, b)(y - b)^2\bigr] + \cdots \]

    \(x - a = \mathrm{d}x\), \(y - b = \mathrm{d}y\) に対して

    • \(0\) 次の項 \(f(a, b)\) は定数
    • \(1\) 次の項は全微分 \(\mathrm{d}f\) に対応
    • \(2\) 次以降の項は高次の微小量を表す

    これは、微分演算子を用いてコンパクトに書ける。

    2変数のテイラー展開（演算子形式）

    \[ f(x, y) = f(a, b) + \left[(x - a)\frac{\partial}{\partial x} + (y - b)\frac{\partial}{\partial y}\right]f + \frac{1}{2!}\left[(x - a)\frac{\partial}{\partial x} + (y - b)\frac{\partial}{\partial y}\right]^2 f + \cdots \quad \cdots \text{①} \]

    ここで

    \[ \left[(x - a)\frac{\partial}{\partial x} + (y - b)\frac{\partial}{\partial y}\right]^2 f = (x-a)^2 \frac{\partial^2 f}{\partial x^2} + 2(x-a)(y-b)\frac{\partial^2 f}{\partial x\,\partial y} + (y-b)^2 \frac{\partial^2 f}{\partial y^2} \]

    であり、偏微分の各項は \((a, b)\) で評価する。

    6.4 3変数・\(n\) 変数への拡張

    3変数の場合

    \[ f(x, y, z) = f(a, b, c) + \left[(x - a)\frac{\partial}{\partial x} + (y - b)\frac{\partial}{\partial y} + (z - c)\frac{\partial}{\partial z}\right]f + \frac{1}{2!}\left[(x - a)\frac{\partial}{\partial x} + (y - b)\frac{\partial}{\partial y} + (z - c)\frac{\partial}{\partial z}\right]^2 f + \cdots \]

    \(n\) 変数の場合

    \[ D = (x_1 - a_1)\frac{\partial}{\partial x_1} + \cdots + (x_n - a_n)\frac{\partial}{\partial x_n} \]

    とおくと

    \[ f(x_1, \ldots, x_n) = f(a_1, \ldots, a_n) + \sum_{k=1}^{\infty}\frac{1}{k!}\,D^k f\big|_{(a_1, \ldots, a_n)} \]

    6.5 導出

    \(f(a + th,\, b + tk)\) を \(t\) の1変数関数とみなす。\(x(t) = a + th\), \(y(t) = b + tk\) とおくと

    前回のパターン①より

    \[ \frac{\mathrm{d}f}{\mathrm{d}t} = \frac{\partial f}{\partial x}\frac{\mathrm{d}x}{\mathrm{d}t} + \frac{\partial f}{\partial y}\frac{\mathrm{d}y}{\mathrm{d}t} = h\frac{\partial f}{\partial x} + k\frac{\partial f}{\partial y} \]
    \[ \frac{\mathrm{d}^2 f}{\mathrm{d}t^2} = \frac{\mathrm{d}}{\mathrm{d}t}\!\left(\frac{\mathrm{d}f}{\mathrm{d}t}\right) = \left[\frac{\partial}{\partial x}\!\left(\frac{\mathrm{d}f}{\mathrm{d}t}\right)\right]\frac{\mathrm{d}x}{\mathrm{d}t} + \left[\frac{\partial}{\partial y}\!\left(\frac{\mathrm{d}f}{\mathrm{d}t}\right)\right]\frac{\mathrm{d}y}{\mathrm{d}t} \]
    \[ = \left[h\frac{\partial^2 f}{\partial x^2} + k\frac{\partial^2 f}{\partial x\,\partial y}\right]h + \left[h\frac{\partial^2 f}{\partial x\,\partial y} + k\frac{\partial^2 f}{\partial y^2}\right]k \]
    \[ = h^2\frac{\partial^2 f}{\partial x^2} + 2hk\frac{\partial^2 f}{\partial x\,\partial y} + k^2\frac{\partial^2 f}{\partial y^2} = \left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right)^2 f \]

    一方、1変数の場合のテイラー展開より

    \[ f(t) = f(0) + f'(0)\,t + \frac{1}{2!}f''(0)\,t^2 + \cdots \]

    \(t = 1\) とおくと

    \[ f(1) = f(a + h,\, b + k) = f(a, b) + \left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right)f + \frac{1}{2!}\left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right)^2 f + \cdots \]

    \(a + h = x\), \(b + k = y\) とおけば式①を得る。

    例題

    例題1（教科書 p.192）

    \(f(x, y) = e^{x+y}\) を \(f(0, 0)\) のまわりに2次まで展開せよ。

    解
    \[ f(0, 0) = e^0 = 1 \]

    1次の偏導関数：

    \[ f_x = e^{x+y} \implies f_x(0, 0) = 1 \]
    \[ f_y = e^{x+y} \implies f_y(0, 0) = 1 \]

    2次の偏導関数：

    \[ f_{xx} = e^{x+y} \implies f_{xx}(0, 0) = 1 \]
    \[ f_{xy} = e^{x+y} \implies f_{xy}(0, 0) = 1 \]
    \[ f_{yy} = e^{x+y} \implies f_{yy}(0, 0) = 1 \]

    したがって

    \[ f(x, y) = 1 + x + y + \frac{1}{2!}[x^2 + 2xy + y^2] + \cdots \]
    \[ \therefore\quad f(x, y) = 1 + x + y + \frac{1}{2}x^2 + xy + \frac{1}{2}y^2 + \cdots \]

    例題2（教科書 p.192）

    \(f(x, y) = \ln(2x - 3y)\) を \(f(2, 1)\) のまわりに2次まで展開せよ。

    解
    \[ f(2, 1) = \ln(4 - 3) = \ln 1 = 0 \]

    1次の偏導関数：

    \[ f_x = \frac{2}{2x - 3y} \implies f_x(2, 1) = 2 \]
    \[ f_y = \frac{-3}{2x - 3y} \implies f_y(2, 1) = -3 \]

    2次の偏導関数：

    \[ f_{xx} = \frac{-4}{(2x - 3y)^2} \implies f_{xx}(2, 1) = -4 \]
    \[ f_{yy} = \frac{-9}{(2x - 3y)^2} \implies f_{yy}(2, 1) = -9 \]
    \[ f_{xy} = \frac{6}{(2x - 3y)^2} \implies f_{xy}(2, 1) = 6 \]

    したがって

    \[ f(x, y) = 0 + 2(x - 2) + (-3)(y - 1) + \frac{1}{2!}\bigl[-4(x-2)^2 + 2 \cdot 6 \cdot (x-2)(y-1) + (-9)(y-1)^2\bigr] + \cdots \]
    \[ \therefore\quad f(x, y) = 2(x - 2) - 3(y - 1) - 2(x - 2)^2 + 6(x - 2)(y - 1) - \frac{9}{2}(y - 1)^2 + \cdots \]

    演習問題

    演習1

    \(f(x, y) = x\sqrt{y}\) を \(f(1, 4)\) のまわりに2次まで展開せよ。

    解答
    \[ f(1, 4) = 1 \cdot \sqrt{4} = 2 \]

    1次の偏導関数：

    \[ f_x = \sqrt{y} \implies f_x(1, 4) = 2 \]
    \[ f_y = \frac{x}{2\sqrt{y}} \implies f_y(1, 4) = \frac{1}{4} \]

    2次の偏導関数：

    \[ f_{xx} = 0 \]
    \[ f_{yy} = -\frac{x}{4y^{3/2}} \implies f_{yy}(1, 4) = -\frac{1}{32} \]
    \[ f_{xy} = \frac{1}{2\sqrt{y}} \implies f_{xy}(1, 4) = \frac{1}{4} \]

    したがって

    \[ f(x, y) = 2 + 2(x - 1) + \frac{1}{4}(y - 4) + \frac{1}{2!}\left[0 + 2 \cdot \frac{1}{4}(x - 1)(y - 4) + \left(-\frac{1}{32}\right)(y - 4)^2\right] + \cdots \]
    \[ \therefore\quad f(x, y) = 2 + 2(x - 1) + \frac{1}{4}(y - 4) + \frac{1}{4}(x - 1)(y - 4) - \frac{1}{64}(y - 4)^2 + \cdots \]

    演習2（教科書 p.193）

    \(f(x, y) = \cos(x - y)\) を \(f(0, 0)\) のまわりに2次まで展開せよ。

    解答
    \[ f(0, 0) = \cos 0 = 1 \]

    1次の偏導関数：

    \[ f_x = -\sin(x - y) \implies f_x(0, 0) = 0 \]
    \[ f_y = \sin(x - y) \implies f_y(0, 0) = 0 \]

    2次の偏導関数：

    \[ f_{xx} = -\cos(x - y) \implies f_{xx}(0, 0) = -1 \]
    \[ f_{yy} = -\cos(x - y) \implies f_{yy}(0, 0) = -1 \]
    \[ f_{xy} = \cos(x - y) \implies f_{xy}(0, 0) = 1 \]

    したがって

    \[ f(x, y) = 1 + \frac{1}{2!}\bigl[-x^2 + 2xy - y^2\bigr] + \cdots \]
    \[ \therefore\quad f(x, y) = 1 - \frac{1}{2}x^2 + xy - \frac{1}{2}y^2 + \cdots \]
